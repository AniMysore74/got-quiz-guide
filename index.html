How to use?

  • Choose a mode of searching - Initials, Hangmen or Anagrams.
  • Choose which databases you want results from - Character names, Objects, Titles, Places, Events
  • Go.

Search formats

  • Initials - No Spaces. Or special characters. Not case sensitive. Example: pO will turn up Perwyn Osgrey
  • Hangmen - All blanks must be underscores _. Spaces will be treated as word seperators.
  • Anagrams- Only Alpabets. Any number of spaces allowed. Not case sensitive.
